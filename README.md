# Emanuele Artom Digital Edition

This project presents a digital edition of selected diary pages by Emanuele Artom, developed as part of a university course in Digital Humanities. The work focuses on the encoding, representation, and web publication of historical textual documents.

## Project Overview

The documents were encoded using XML-based markup and presented through a web interface built with HTML, CSS, and JavaScript. The goal of the project was to make the encoded text accessible and navigable while preserving relevant textual and structural information from the original documents.

## Technologies

* XML
* HTML5
* CSS3
* JavaScript

## Features

* Digital representation of historical diary pages
* Structured text encoding
* Web-based interface for consultation
* Navigation between encoded documents and project sections

## Academic Context

This project was developed during my Bachelor's degree in Digital Humanities at the University of Pisa.
